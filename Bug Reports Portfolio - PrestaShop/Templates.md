# Bug Report Templates

This folder contains reusable, professional bug report templates you can use across all projects.

---

## **1. Simple Bug Report Template**

```
Title:
Module/Area:
Environment:
Severity:
Priority:

Steps to Reproduce:
1.
2.
3.

Expected Result:

Actual Result:

Attachments:
```

---

## **2. Detailed Bug Report Template (Industry Standard)**

```
# Bug Report

**ID:** BR-XXX  
**Title:**  
**Module / Feature:**  
**Environment:** (Browser, OS, URL, Version)  
**Severity:**  
**Priority:**  

---

## Steps to Reproduce
1. 
2. 
3. 

---

## Expected Result

---

## Actual Result

---

## Additional Notes
- Browser console errors:
- Network errors:
- Logs (if any):

---

## Attachments
- Screenshot(s)
- Video (if any)
```

---

## **3. Screenshot Naming Convention Template**

```
project_name/module/bug_id_short_description.png

Examples:
prestashop/cart/BR-001-broken-image.png
prestashop/checkout/BR-014-zero-quantity-checkout.png
```
