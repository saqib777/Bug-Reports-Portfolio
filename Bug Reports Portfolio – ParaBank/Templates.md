# ParaBank Bug Report Templates

This folder contains standardized templates for documenting defects found during testing of the ParaBank application. These templates are designed to maintain clarity, consistency, and professional QA formatting across all reports.

---

## Simple Bug Report Template

```
Title:
Module/Area:
Environment:
Severity:
Priority:

Steps to Reproduce:
1.
2.
3.

Expected Result:

Actual Result:

Attachments:
```

---

## Detailed Bug Report Template

```
Bug ID:
Title:
Module / Feature:
Environment: (Browser, OS, Version, URL)
Severity:
Priority:

Steps to Reproduce:
1.
2.
3.

Expected Result:

Actual Result:

Notes:
- Console logs (if any):
- Network errors (if any):
- Additional observations:

Attachments:
- Screenshots
- Videos (if any)
```

---

## Field Validation Template

```
Field Name:
Validation Type:
Input Tested:
Expected Behavior:
Actual Behavior:
Status:
Notes:
```

---

## Screenshot Naming Convention

```
parabank/module/bug_id_short_description.png

Examples:
parabank/login/BR-001-invalid-login.png
parabank/transfer/BR-008-negative-amount.png
```
